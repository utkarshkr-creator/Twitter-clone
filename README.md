# Requirements
user should be able to post a tweet.
## Tweet

- content (250 char MAX)
- likes
- noOfRetweets
- comment 
- hashtages(#)
- user


## Like

- like on tweet, like on comment tweet
- user

## Comment
- content
- like 
- user

## User

- username
- noOfFollowers
- Email/passwords
- List of tweets
- bio

## HashTag
- word
- List of tweets
- ranking
